# EAT MT 游戏

**在毒气森林中的生存挑战**

## 故事背景

你被困在一片神秘的毒气森林中，四周弥漫着致命的毒雾。在这片被诅咒的土地上，只有一种神奇的食物能够抵御毒气的侵蚀——**魔法馒头**。

传说这些馒头是由古代森林守护者留下的，蕴含着强大的生命力。你必须在这片充满危险的森林中不断寻找这些珍贵的馒头，维持自己的生命，寻找逃生的希望。

每一秒，毒气都在侵蚀你的身体，只有收集到足够的魔法馒头，你才能在这片死亡森林中存活下去。时间就是生命，每一次呼吸都可能是最后一次...

## 游戏概述

EAT MT是一款基于Three.js开发的生存冒险游戏，玩家需要在充满毒气的森林中控制角色移动，通过收集随机生成的魔法馒头来维持生命值。游戏采用俯视视角，具有网格移动机制和生命值系统，营造紧张刺激的生存体验。

## 游戏玩法

### 基本操作
- **左键点击**：向点击位置移动角色
- **右键点击**：停止当前移动
- **自动移动**：角色会自动向目标位置移动

### 游戏机制

#### 1. 网格移动系统
- 游戏地图采用50x50像素的网格系统
- 角色每次移动都会对齐到网格中心
- 最大移动距离限制为5个网格

#### 2. 毒气侵蚀系统
- 初始生命值：100点
- 毒气每秒侵蚀1点生命值
- 当生命值降至0时，角色被毒气吞噬，游戏结束

#### 3. 魔法馒头收集
- 森林中会随机生成魔法馒头
- 最多同时存在20个魔法馒头
- 魔法馒头会在玩家周围10个网格范围内随机生成
- 每个魔法馒头提供+5点生命值恢复，暂时抵御毒气侵蚀

#### 4. 角色状态
- **站立状态**：角色静止时显示站立动画
- **奔跑状态**：角色移动时显示奔跑动画
- **死亡状态**：生命值为0时显示死亡动画

#### 5. 游戏结束与重新开始
- 当生命值降至0时，游戏结束并显示统计界面
- 统计信息包括：生存时间、收集馒头数量、最终得分
- 点击"重新开始"按钮可以重新开始游戏
- 重新开始会重置所有游戏状态，包括角色位置、生命值和馒头

## 游戏界面

### 主界面元素
- **游戏画布**：中央的游戏区域，显示角色和馒头
- **坐标显示**：右上角显示玩家当前网格坐标
- **生命值显示**：显示当前生命值
- **馒头计数**：显示当前地图上的馒头数量
- **游戏Logo**：左上角显示游戏标题和图标
- **游戏结束界面**：游戏结束时显示统计数据，包含生存时间、收集馒头数量和最终得分

### 视觉元素
- **草地背景**：整个地图覆盖草地纹理
- **网格线**：半透明的白色网格线辅助定位
- **角色动画**：站立、奔跑、死亡三种状态
- **馒头图标**：可收集的食物道具
- **雾气效果**：漂浮的白色雾气粒子，营造毒气森林氛围

## 游戏目标

在毒气森林中尽可能长时间地生存下去，通过不断收集魔法馒头来抵御毒气侵蚀，创造生存记录。每一次游戏都是新的挑战，看看你能否在这片死亡森林中存活更久！

## 技术特点

- 使用Three.js进行3D渲染
- 基于网格的移动系统
- 实时碰撞检测
- 自动相机跟随
- 响应式界面设计

## 运行方式

1. 确保所有文件在同一个目录下
2. 使用现代浏览器打开 `index.html` 文件
3. 游戏将自动开始
